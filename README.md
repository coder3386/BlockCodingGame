# BlockCodingGame
캡스톤 디자인 1 블럭코딩게임 개발용


요구사항

-오브젝트 : 플레이어 캐릭터, 도착지

- 플레이어 캐릭터가 있어야 한다.
- 플레이어 캐릭터는 상(점프) 좌 우 기능을 수행 할수 있어야 한다.
- 플레이어 캐릭터의 목표는 함정을 피해 도착지에 도착할수 있어야 한다.

- 플레이어 캐릭터의 움직임을 유저가 직접 만들수 있어야 한다.
- 플레이어 캐릭터의 움직임을 할수 있게 하는 코드 블럭이 존재해야한다.

- 코드블럭은 반복문 조건문의 기능을 할수 있어야한다.
- 코드블럭은 캐릭터의 이동을 조작할수 있어야한다.
- 코드블럭은 유저가 직접 얻어야한다.

- 코드블럭 사용방법을 알려주는 튜토리얼이 존재해야한다.
